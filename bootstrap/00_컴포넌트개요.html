컴포넌트 개요 정리하기

        컴포넌트는 특정 기능이나 기능 집합을 의미
        독립적으로 설계된 기능, 재사용이 가능한 모듈 또는 단위
        객체, 기능(함수),메서드 등

Button

        클릭 가능한 버튼을 생성
        부트스트랩으로 버튼을 시작할 때
        class 내부에 btn 버튼 시작기능을 작성해주고
        추가로 작성해주고 싶은 스타일을 적용
    
        input type에 있는 버튼기능을 사용하길 원한다면 아래와 같이 사용
    
        a 태그를 버튼처럼 사용하길 원한다면 아래와 같이 사용
a태그버튼

버튼 테두리 표시하기

버튼크기 조정

        버튼의 크기는 btn-lg나 btn-sm와 같은 크기조정을 사용 적용
        padding을 활용해서 버튼의 크기를 조정할 수 있음

버튼 비활성화

        버튼을 비활성화 하기 위해 disabled 를 추가해주면 비활성화 가능

        a태그 버튼을 비활성화하기 위해서는 aria-disabled="true"를 넣어주면 됨
        disabled 도 같이 사용해줘야함
