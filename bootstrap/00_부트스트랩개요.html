        추가적인 스타일이 필요할 경우 css를 사용해서 디자인을 만들어줘야함

        integrity 이란?
            - 링크를 안전하게 가져오기 위해 설정하는 속성 (필수는 아니지만 보안,신뢰성을 위해 사용하는 것이 좋음)
        crossorigin 이란?
            - anonymous 자격 증명을 포함하지 않도록 설정 (필수는 아니지만 보안,신뢰성을 위해 사용하는 것이 좋음)

라이브러리와 프레임워크의 차이

        라이브러리 vs 프레임워크

        라이브러리 = 샤브샤브뷔페
            -> 개발자가 개발을 하는데 있어 사용범위가 더 넓음

        프레임워크 = 샤브샤브밀키트
            -> 개발자가 개발을 하는데 있어 사용범위가 제한적임

class 사용 방법 정리

        container : 반응형 고정 폭을 제공하는 역할
                    페이지 콘텐츠를 정렬하고 중앙에 배치
                    기본적으로 페이지의 양 옆에 여백을 줘서 콘텐츠를 중앙에 정렬
        row       : container에서 가로 한 줄을 나타냄
                    가로 한줄은 col-12로 이루어져있음
                    가로 한 줄의 col이 12를 초과할 경우, 다음 줄로 자동 이동
        col       : row(가로 한 줄)에서 세로로 몇 칸을 차지할 것인지 표현
                    col-1 : 세로로 중간사이즈로 1 칸
                    col - xs : 세로로 가장 작은 사이즈로 1칸 차지
                    col-4 : 가로 한줄에서 4칸을 차지하게 되며
                            가로 한줄은 총 12칸이기 때문에 여유칸은 8칸이 됨
        
        
 
            
 
            

            

        

        
        
 
            
 
            

        


        card : 부트스트랩에 있는 카드 스타일을 적용
        card-body : 카드 본문 틀 안에 이미지나 타이틀이나 텍스트 등을 넣어줌
        card-title : 카드 본문에서 제목을 나타냄
                    제목 스타일을 적용
        card-text : 카드에서 내용을 작성하는 공간
                    텍스트 스타일을 적용
        card-img-위치 : card에 있는 img 위치를 적용
                        img가 카드 바깥으로 나가는 것을 방지
                        카트 틀 안에 이미지가 있을수 있도록 도와줌
                        width / height를 사용하지 않아도됨
                        card-img-top    : 이미지를 위에 배치
                        card-img-bottom : 이미지를 아래에 배치
        
        img 설정하기
            img-fluid : col이나 특정 가로 크기에 맞게 조정
                col-4 img-fluid
                4칸 기준으로 이미지 크기가 딱 맞게 들어가짐
                width : 100% , height : auto
            
            img-thumbnail : 이미지에 테두리와 패딩을 추가해서 썸네일 이미지 스타일 느낌을 적용
            
            rounded : 이미지의 모서리를 둥글게 만듬
                            border-radius : 25rem
            
            round-circle : 이미지를 가볍게 원형으로 만듬
                            border-radius : 50%

            round-pill : 이미지 모서리를 완전히 둥글게 만듬
                            border-radius : 50rem
