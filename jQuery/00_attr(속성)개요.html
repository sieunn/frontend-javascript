attr

            attr(attribute) 속성
            jQuery에서 html 태그 속성을 가져오거나 설정할 때 사용되는 기능

            -제이쿼리를 작성하지 않았을 때 속성 사용방법
            getAttribute = 속성 정보 가져오기
            getAttribute('원하는속성선택')

            setAttribute = 속성 정보 변경하기
            setAttribute('변경원하는속성선택', '변경할값작성')


            -제이쿼리를 사용한 속성사용방법
            attr만 작성해주면 끝
            atrr('원하는속성선택')
            atrr('변경원하는속성선택', '변경할값작성')

            예제
            
            
            src, alt style, class 모두 attr에 해당
            attr은 태그 안에 작성된 속성값을 거져오거나 변경하거나 제거할 때 사용

            1. 태그 안에 있는 속성값 가져오기
                $("img").attr('src'); // src 안에 있는 그림 주소값을 가져오는 것
            2. 태그 안에 있는 속성값 변경하기
                $("img").attr('src', '변경할주소값설정') 
                //src안에 기존에 존재했던 주소를 변경할주소값설정으로 변경
            3. 태그 안에 있는 속성값을 여러개 변경하기
                $("img").attr( {  src:'변경할주소값' , alt:"변경된이미지설명"  } )
